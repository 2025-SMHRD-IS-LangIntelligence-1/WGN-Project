### 와구냠(Wagunyam) 팀명 : 개발맛집
*** 
## 서비스 개요
**감성 분석, 추천 알고리즘 기반 광주 맛집 추천 소셜 네트워크 서비스**

## 서비스 소개

와구냠(Wagunyam)은 사용자 리뷰의 감성을 분석하고 추천 알고리즘을 적용해, 광주의 숨은 맛집을 쉽고 재미있게 발견할 수 있는 소셜 네트워크 서비스입니다.
리뷰 속 키워드를 시각적으로 보여주고, 나와 취향이 비슷한 사람들의 맛집을 추천받을 수 있으며, 카카오맵과 연동된 나만의 맛집 지도를 만들 수도 있습니다. 단순한 맛집 검색을 넘어, 좋아요·팔로우·댓글 등 소셜 기능을 통해 사람들과 경험을 공유하는 맛집 커뮤니티 플랫폼을 지향합니다.

## 프로젝트 기간
2025.07.30 ~ 2025.08.18 (14일)

## 주요 기능

- 사용자 리뷰 감성 분석 및 핵심 키워드 워드클라우드 시각화 기능 (FastText 활용)
- 유사도 분석 기반 음식점 및 피드 검색 기능 (TF-IDF, BERT 활용)
- 사용자 데이터 기반 맞춤형 맛집 피드 추천 기능
- 나만의 맛집 지도 만들기 기능 (KakaoMap API 활용)
- 팔로우, 좋아요, 댓글, 알림 등 SNS 기능

## 기술 스택

- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot
- Database: MySQL
- NLP: FastText, TF-IDF, BERT
- 외부 API : Kakaomap, Cloudinary
